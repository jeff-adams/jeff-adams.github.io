console space trading/economy game prototype

View on GitHub

This project was a pair programming project that was completed over a two week period, for the Microsoft Software & Systems Academy course. The purpose was to learn to design and code as part of a small team.

We designed the project using a Model-View-ViewModel approach. I worked most prevalently on the View; creating the menu state machine to capture user input, the ship's HUD which is updated via events, and the multiple viewscreens to display different information.

Please check out the wiki for more details of the project requirements as well as supporting documentation of the build process.

console based long list tracking application

View on GitHub

This application was created for the Microsoft Software & Systems Academy course. The purpose was to learn to develop an application from a user's requirements and utilize git for version control.

I designed the application so that the data and logic was decoupled in a seperate class library. I took what I had learned about making a console based menu UI from previous projects and created a flexible graphical UI to interact with a list of tasks. I had previously had a problem with screen flashing when I was updating the entire screen full of data. I solved this problem by only redrawing the previous line of data and the next, highlighted line of data.
